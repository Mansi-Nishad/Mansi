HTML Tags

  1. <html>: The root element that wraps all content on the entire page.
  2. <head>: Contains meta-information about the HTML document, such as title, character set, etc.
  3. <title>: Defines the title of the HTML document, displayed in the browser's title bar.
  4. <body>: Contains the content of the HTML document, including text, images, links, etc.
  5. <h1> to <h6>: Headings that define the structure of the document. <h1> is the largest, and <h6> is the smallest.
  6. <p>: Represents a paragraph of text.
  7. <a>: Creates a hyperlink, linking to another page or resource.
  8. <img>: Embeds an image in the HTML document.
  9. <ul>: Defines an unordered list.
  10. <ol>: Defines an ordered list.
  11. <li>: Represents a list item in both ordered and unordered lists.
  12. <div>: A generic container element used for layout and grouping other HTML elements.
  13. <span>: A generic container inline element used for styling a specific portion of text.
  14. <strong>: Represents strong importance or importance.
  15. <em>: Represents emphasized text.
  16. <br>: Represents a line break within text.
  17. <hr>: Represents a thematic break or horizontal rule.
